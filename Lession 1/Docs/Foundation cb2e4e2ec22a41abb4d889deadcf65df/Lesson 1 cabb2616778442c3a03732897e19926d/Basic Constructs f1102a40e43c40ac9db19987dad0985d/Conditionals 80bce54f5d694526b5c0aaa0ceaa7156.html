Conditionals

Structures utilized to branch logic

If statement

If this, then that !

Structure

structure type
 ‚ñº     ‚ñº condition needs to equal 'true', for body to execute
if ( true )
{
    // code ...               ‚ùÆ- Body of statement
}

Example: 1

if ( 1 < 2 )         // is 1 less than 2
{
    printf ( "one is less than two !" );
}

Example: 2

if ( 3 < 2 )           // is 3 less than 2
{
    printf ( "one is less than two !" );
}

Example: 3 

if ( true )   // Forcing conditional with value
{
    printf ( "one is less than two !" );
}

If / Else statement

If this, that, else, that other thing !?

Structure

if ( conditional_statement )
{
    // code ...        ‚ùÆ- Body of 'true' statement
}
else
{
    // code ...        ‚ùÆ- Body of 'false' statement
}

Example: 1

if ( 1 != 1 )          // is 1 not equal to 1
{
    printf ( "condition is true !" );
}
else
{
    printf ( "condition is false !" );
}

Example: 2

if ( 0 )            // 0 is equal to false
{
    printf ( "condition is true !" );
}
else
{
    printf ( "condition is false !" );
}

Short forms

It is possible to write if and if / else statements, in short form ‚Ä¶ or without curly braces !

Example: 1

if ( 100 < 200 )
  if ( 300 < 200 )

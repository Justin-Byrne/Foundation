For loop

Automatically repeats and increments until a condition is met

Structure

for ( init; condition; increment ) 
{
    // code ...
}

init: allows you to declare and set a loop control variable

Example: define initializer variable

{
    // code ...
}

Here, weâ€™ve declared an index variable ( named i ), that weâ€™ll use for loop control

Example: define conditional statement

{
    // code ...
}

Here, weâ€™ve set the condition to be evaluated. Once i is no longer less than 5, the body of the code will no longer execute, and the program will exit the for loop.

Example: define increment

{
    // code ...
}

Here, weâ€™ve set how our index variable ( i ) should increment after each loop.

		printf ( "Loop: %d \n", i );

Break

Letâ€™s break some loops !

break statement breaks out of the current loop, and continues the rest of the program

Example:

for ( int i = 0; i < 5; i++ ) 
{
    if ( i == 2 )  
    {
				break;       // use break to terminate loop execution
    }

    printf ( "i: %d \n", i );
}

Prints out:

i: 0 
i: 1

This only prints out two twice, because after the internal if statement confirms that i is equal to 2, it breaks out of the for loop.

This only breaks out of the nearest nested loop !

## Installation

Migrate to your desired download location (via your shell/terminal) and download this repository

```sh
git clone https://github.com/Justin-Byrne/Foundation.git
```

Navigate to the `Makefile` 

Type:

```sh
> make
```

## Usage

### Linux/Mac

```sh

```
